# 職業・資格紹介サイト

## サイト概要
職業や資格などを紹介する。
性格診断からその人に合った職業やオススメの資格などを紹介する。

### サイトテーマ
世の中にどういう仕事や資格があるか知る。
自分の可能性を気付くこと。

### テーマを選んだ理由
自分がやりたいことがわからない人や、現状に満足していない若い人は多いと感じるので、自己分析を行うきっかけがあれば良いと思いテーマを選んだ。

### ターゲットユーザ
10代から20代の自分の進路に迷っている人
将来に不安を抱えてる人
### 主な利用シーン
これからの進路を迷っている時や、現在の進路に不安を抱えている時。
